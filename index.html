The embed script tag should be included in partner webiste HTML like this:

<script id="everfi-program-content-iframer"
        src="https://everfi-next-dev.net/embed_script.js"></script>
  • a global object "everFiProgramContentIframeOptions" is used to set parameters
  • data attributes on the script tag are used to override values in the src URL
  • query parameters in the src URL are used if not provided by either of the first two methods
  • module can be provided as the ID or as the Next content slug
  • additional params for the URL can be added in any of the three spots where account/program/playlist/module are set
  • and will be added as query string parameters to the embed URL
  • account, program, playlist, and module are all required
  • "fidev-next.net" should be replaced with the desired next environment domain ( either "everfi-next-dev.net", "everfi-next-staging.net", or "everfi-next.net" )
